# fastcalc

![](https://travis-ci.org/rekols/fastcalc.svg?branch=master)

мощный алгоритм калькулятора.

![](screenshot/20181220103800.png)

## зависимости

* sudo apt install qt5-default g++

## Как запустить ?

* mkdir build

* cd build

* qmake ..

* make

## Применение

* ./fastcalc
